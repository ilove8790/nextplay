# NextPlay

동영상 자동 재생 플레이어

## 다운로드

<https://github.com/ilove8790/nextplay/archive/refs/heads/main.zip>

설치할 임시 폴더에 저장(파일명: nextplay-main.zip)하고 압축해제 합니다.

## 설치

Chrome Extension 에 개발자 모드로 설치합니다.

1. Chrome 에서 우측 상단의 (...)을 클릭
2. 메뉴에서 '확장프로그램' 선택 --> 확장프로그램 관리 선택
3. 확장 프로그램 화면에서 우측 상단의 '개발자 모드'를 켭니다.
4. 좌측 상단의 '압축해제된 확장프로그램 로드'를 선택
5. 다운로드 받은 'nextplay-main' 폴더 아래 'extensions' 폴더까지 진입후 '폴더 선택'을 클릭합니다.
6. 확장 프로그램 화면에 추가된 'NextPlay'에서 '세부정보'를 선택
7. 세부정보 옵션 중 '툴바에 고정'을 켜면 'N'자 아이콘이 브라우저 상단에 고정됩니다.
8. 'N'자 아이콘을 클릭하면 'Enable Auto Play'를 On/Off 할 수 있습니다.

## 사용

1. 처음 동영상 재생 화면으로 진입하면 자동으로 동영상이 재생됩니다.
2. 동영상이 끝나면 자동으로 다음 동영상이 재생됩니다.
3. 모든 동영상 시청이 끝나면 평가 화면에서 멈추게 됩니다.
4. 크롬 브라우저 상단의 'N'자 아이콘을 클릭해서 'Enable Auto Play'를 Off 하면 끝.
